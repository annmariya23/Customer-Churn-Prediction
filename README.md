# Customer Churn Prediction

## Project Overview
This project predicts whether a customer is likely to leave a telecom service using Machine Learning techniques.

The project includes:
- Data Cleaning
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Machine Learning Models
- Performance Evaluation

---

## Dataset
Telco Customer Churn Dataset

Dataset Source:
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost
- Jupyter Notebook

---

## Machine Learning Models Used
1. Logistic Regression
2. Random Forest Classifier
3. XGBoost Classifier

---

## Evaluation Metrics
- Accuracy
- Classification Report
- ROC-AUC Score
- Confusion Matrix

---

## Exploratory Data Analysis (EDA)
The following visualizations were created:
- Customer Churn Distribution
- Correlation Heatmap
- Monthly Charges vs Churn
- Tenure vs Churn
- Feature Importance Graph

---

## Project Workflow
1. Import Libraries
2. Load Dataset
3. Data Cleaning
4. Handle Missing Values
5. Encode Categorical Variables
6. Perform EDA
7. Train Machine Learning Models
8. Evaluate Models
9. Compare Model Performance

---

## Results
The models successfully predicted customer churn.

Key findings:
- Customers with high monthly charges are more likely to churn.
- Contract type and tenure strongly affect churn.
- XGBoost and Random Forest achieved strong prediction performance.

---

## Conclusion
This project demonstrates how Machine Learning can help businesses identify customers who may leave their services and improve customer retention strategies.
